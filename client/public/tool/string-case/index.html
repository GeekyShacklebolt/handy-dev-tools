Frequently Asked Questions

What string cases are supported?

camelCase, PascalCase, snake_case, kebab-case, SCREAMING_SNAKE_CASE, dot.case, path/case, Title Case, Sentence case, UPPER CASE, lower case, and more.

When should I use each case?

camelCase for JavaScript variables, PascalCase for classes/components, snake_case for Python/Ruby/database columns, kebab-case for CSS classes and URL slugs, SCREAMING_SNAKE_CASE for constants.

Can it handle acronyms correctly?

Yes, the converter intelligently handles acronyms like 'HTTP' or 'API'. For example, 'httpAPIRequest' correctly converts to 'http_api_request' in snake_case.
