Frequently Asked Questions

What is PHP serialize format?

PHP serialize() converts a PHP value to a storable string representation. For example, an array ['a', 'b'] becomes a:2:{i:0;s:1:"a";i:1;s:1:"b";} with type and length information.

Where is PHP serialized data commonly found?

WordPress stores plugin settings, widget data, and transients in serialized format. It is also used in PHP session files and legacy database storage.

Is PHP unserialize safe?

Unserializing untrusted data can be dangerous as it can trigger object instantiation and magic methods. This tool safely parses the data format without executing PHP code.
